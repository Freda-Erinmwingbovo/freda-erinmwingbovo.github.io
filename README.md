## Overview

A professional data analytics and data science portfolio showcasing end-to-end analytical projects across six industries, retail, e-commerce, banking, HR, supply chain, and clinical healthcare. Every project follows a rigorous analytical workflow from raw data ingestion through to machine learning, visualisation, and actionable business or clinical recommendations.

The portfolio is built as a single-page HTML application with a custom dark UI, hosted on GitHub Pages.
